Detailed Description

A manager for ScriptComponents with type-based priority.

  • Deferred additions if added during invocation (stored in pendingAdd).
  • Removals set comp=null if we are in the middle of invocation, or remove them directly if not in invocation.
  • After invocation, we remove any null comps and also insert any pendingAdd comps, then we re-sort.
