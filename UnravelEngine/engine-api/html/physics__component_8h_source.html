124enum class force_mode : uint8_t
125{
126 // Interprets the input as torque (measured in Newton-metres), and changes the angular velocity by the value of
127 // torque * DT / mass. The effect depends on the simulation step length and the mass of the body.
128 force,
129
130 // Interprets the parameter as angular acceleration (measured in degrees per second squared), and changes the
131 // angular velocity by the value of torque * DT. The effect depends on the simulation step length but does not
132 // depend on the mass of the body.
133 acceleration,
134
135 // Interprets the parameter as an angular momentum (measured in kilogram-meters-squared per second), and changes the
136 // angular velocity by the value of torque / mass. The effect depends on the mass of the body but doesn't depend on
137 // the simulation step length.
138 impulse,
139
140 //: Interprets the parameter as a direct angular velocity change (measured in degrees per second), and changes the
141 //: angular velocity by the value of torque. The effect doesn't depend on the mass of the body and the simulation
142 //: step length.
